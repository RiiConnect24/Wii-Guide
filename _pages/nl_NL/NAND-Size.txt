Een geldige NAND backup zou een van de volgende formaten moeten hebben.
{: .notice--primary}

Soms krijgt u een back-up met een ander formaat dan één van deze lijst. Het is prima dat een back-up groter dan verwacht word, zorg er echter voor dat er nooit een backup gebruikt word * kleiner * dan verwacht aan de hand van de onderstaande Nand grotes.
{: .notice--info}

Let op: een *min size* NAND backup zal altijd zo klein mogelijk zijn om ruimte te besparen, en matcht waarschijnlijk niet in grote met andere NAND backups van je 3DS. Dit is prima, maar zorg er voor dat het overeenkomt met een van de formaten op deze pagina.
{: .notice--info}

De merknaam verwijst naar de fabrikant van de NAND chip in je 3ds, **niet** je sd kaart.
{: .notice--warning}

** De back-up moet overeenkomen met een van de formaten op deze pagina; Als dit niet het geval is, moet je deze verwijderen en een nieuwe maken! **
{: .notice--danger}

## 2DS:    

    + Toshiba NAND:     
      988.807.168 bytes = 943 MiB    

    + Toshiba NAND:    
      1.979.711.488 bytes = 1.84 GiB    

    + Samsung NAND:    
      1.000.341.504 bytes = 954 MiB    

    + Samsung NAND:    
      1.300.234.240 bytes = 1.21 GiB    

## Old 3DS:    

    + Toshiba NAND:     
      988.807.168 bytes = 943 MiB    

    + Samsung NAND:    
      1.000.341.504 bytes = 954 MiB    

## New 3DS:    

    + Toshiba NAND:    
      1.979.711.488 bytes = 1.84 GiB    

    + Samsung NAND:    
      1.300.234.240 bytes = 1.21 GiB    

    + Samsung NAND:
      1.954.545.664 bytes = 1.82 GiB
