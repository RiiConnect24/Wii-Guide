#### Istruzioni

##### Sezione I - Preparazione

1. Installa WAMP nella cartella predefinita
1. Copia il contenuto dell'archivio `3ds-private-update-server.zip` nella cartella `C:\wamp\www\` (sovrascrivi i file già esistenti)
1. Copia il file `update.php` nella cartella `C:\wamp\www\` (sovrascrivi i file già esistenti)
1. Copia la cartella`updates` presente all'interno dell'archivio `.zip` per il downgrade alla versione 9.2.0 nella cartella `C:\wamp\www\`
1. Sposta il file `node.exe` nella cartella `C:\wamp\www\updates`
1. Sposta il file `C:\wamp\www\CdnCiaUnpack.js` nella cartella `C:\wamp\www\updates`
1. Nella cartella `C:\wamp\www\updates`, trascina il file `CdnCiaUnpack.js` dentro l'applicazione `node.exe` per creare il file `GetSystemUpdate.partial.xml`
1. Sposta il file `node.exe` e il file `CdnCiaUnpack.js` nella cartella `C:\wamp\www\`
1. Copia il file `GetSystemUpdate.partial.xml` nella cartella `C:\wamp\www\` (così che esista una copia del file sia nella cartella `C:\wamp\www\updates` che nella cartella `C:\wamp\www\`)
1. Clicca sull'icona di WAMP nella [system tray](https://it.wikipedia.org/wiki/System_tray), quindi clicca su "Put Online"
1. Apri il command prompt e digita il comando `ipconfig /all` per ottere l'indirizzo IPv4 del tuo computer, ti servirà a breve
1. Copia il file `ntr.bin` nella directory principale della tua scheda SD
1. Copia il _contenuto_ della cartella `starter` dell'archivio `starter.zip` nella directory principale della tua scheda SD
1. Inserisci la scheda SD nel tuo 3DS

##### Sezione II - Aggiornamento

1. Avvia il 3DS in modalità di ripristino tenendo premuto (L)+(R)+(A)+(Su) durante l'accensione
1. Rifiuta l'aggiornamento e riavvia la console
1. Avvia Cubic Ninja (se ha Ninjhax già installato, tieni premuto (L)+(R)+(X)+(Y) per cancellarlo)
1. Seleziona "Create", quindi "QR Code" and infine "Scan QR Code"
1. Scannerizza il codice QR appropriato per la tua regione presente all'interno dell'archivio `.zip` di NTR CFW
1. Dal menu HOME, premi contemporaneamente (Y) e (X) per aprire il menu di NTR
1. Abilita il debugger e chiudi il menu
1. Usa la pagina web di configurazione del tuo router per ottenere l'indirizzo IP della tua console (cerca su google se non sai come fare, ogni router è differente)
1. Avvia NTR Debugger dal tuo computer
1. Immetti i seguenti comandi (sostituisci XXX.XXX.XXX.XXX con l'indirizzo IP della tua console e YYY.YYY.YYY.YYY con l'indirizzo IP del tuo computer)    
    + `connect("XXX.XXX.XXX.XXX", 8000)`
    + `write(0x15E424, tuple(map(ord, "http://YYY.YYY.YYY.YYY/update.php\0")), pid=0x25)`
    + `write(0x15E0EC, tuple(map(ord, "http://YYY.YYY.YYY.YYY/update.php\0")), pid=0x25)`
    + `write(0x15E463, tuple(map(ord, "http://YYY.YYY.YYY.YYY/update.php\0")), pid=0x25)`
1. Aggiorna la console andando su Impostazioni della console, poi "Impostazioni generali", infine scorri a destra e seleziona "Aggiornamento"
