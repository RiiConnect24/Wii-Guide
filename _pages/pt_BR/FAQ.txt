<a name="faq_latestfw" />**P:** *Estou na última versão do sistema, meu 3DS é hackeável?*    
**R:**Sim, a 11.2 é hackeável usando o método do [Homebrew Launcher (Soundhax)] (https://3ds.guide/homebrew-launcher-(soundhax))

<a name="faq_updatecfw" />**P:** *Como eu atualizo o Luma3DS?*    
**R:** Use o Luma3DS Updater

<a name="faq_gatewaysky" />**P:** *Eu ouvi falar sobre essa coisa que tenho de comprar (Gateway, Sky3DS, etc), isso é algo que eu preciso?*    
**R:** Não. O arm9loaderhax é o método superior para hax que roda bem cedo na inicialização do sistema e abre a oportunidade para coisas como Custom Firmwares de verdade.

<a name="faq_gatewaysaves" />**P:** *Como eu transfiro saves do Gateway para os jogos instalados no sistema?*    
**R:** Veja [isto] (https://gbatemp.net/threads/425743/)

<a name="faq_need" />**P:** *Do que eu preciso para este guia?*    
**R:** Este guia pode ser completado utilizando apenas software livre

<a name="faq_risky" />**P:** *O quão arriscado é hackear meu console?*    
**R:** Bricks agora são *basicamente* impossíveis a menos que você ignore/pule quaisquer verificações de segurança.

<a name="faq_homebrew" />**P:** *Posso rodar homebrews e emuladores incríveis com isso?*    
**R:** Sim! O arm9loaderhax junto a uma Custom Firmware lhe dá acesso não só ao Homebrew Launcher, como também a hacks em todas as versões, o que significa que você o mantém permanentemente, mesmo que atualize.

<a name="faq_regionfree" />**P:** *Posso usar isso para jogar jogos de outras regiões?*    
**R:** O Luma3DS suporta patches de Region Free, o que permite aos usuários instalar jogos de qualquer região sem modificações ou etapas extras.

<a name="faq_updates" />**P:** *É seguro atualizar a SysNAND com CFW?*    
**R:** Sim. Isso é seguro, parem de perguntar sobre isso.

<a name="faq_support" />**P:** *Onde devo buscar suporte?*    
**R:** Para suporte, peça ajuda no [#3dshacks on Rizon IRC](https://www.reddit.com/r/3dshacks/wiki/irc) ou [3DS Hacking on Discord](https://discord.gg/MWxPgEp). Ambos os canais de suporte são exclusivamente em inglês.  

<a name="faq_le4gbsd" />**P:** *Posso fazer isso com um cartão SD de 4GB (ou menor)?*    
**R:** Graças ao novo método via CTRNAND, sim!

<a name="faq_ge128gbsd" />**P:** *Posso usar um cartão SD de 128 GB (ou maior)?*    
**R:** Pode, mas alguns usuários relatam lentidão usando cartões SD desse tamanho, independente da classe do cartão. Note que o 3DS não funciona com cartões SD formatados em exFAT; você precisa formatá-lo em FAT32.

<a name="faq_movesd" />**P:** *Como mudo pra um cartão SD novo no final disso?*    
**R:** Apenas copie todos os arquivos do cartão SD antigo para o novo.

<a name="faq_NNID" />**P:** *Posso manter minha NNID?*    
**A:** If you start with an NNID and follow every step, you will end up keeping your NNID at the end.

<a name="faq_systransfer" />**P:** *E as transferências de dados?*    
**A:** Uma vez que você tenha completado o guia, você pode fazer transferências de dados de e para 3DSs ou 2DSs com arm9loaderhax da mesma forma que faria num console normal ou EmuNAND (você perderá quaisquer jogos/temas/DLCs não legítimos na transferência; os saves permanecerão mesmo para os jogos perdidos).

<a name="faq_nopc" />**P:** *Posso fazer isso sem um computador (ex. com um celular Android)?*    
**R:** Pode, tudo que você precisa é a habilidade de colocar os arquivos no cartão SD!

<a name="faq_problem" />**P:** *Socorro! Algo ruim aconteceu e não consigo ligar o 3DS...*    
**R:** Por favor, consulte o [guia de troubleshooting](troubleshooting).
