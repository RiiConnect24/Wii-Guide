Todos os cartões de jogo contêm uma cópia da versão de sistema mínima para poder jogar o jogo, permitindo assim a qualquer pessoa que se encontre numa versão inferior usar o cartão de jogo para atualizar o seu sistema (sem precisar de ligação à Internet).
{: .notice--info}

Cart Updates will only deliver updates to core features, such as the System Settings, Home Menu, etc. Cart Updates will not deliver updates to Nintendo 3DS Sound and Network features, such as System Transfer, Internet Browser, StreetPass Mii Plaza or eShop.
{: .notice--info}

This means that a cart update from a version <9.0.0 will break [Soundhax](homebrew-launcher-(soundhax))! You will need an [alternate method](homebrew-launcher-(alternatives)) of entering the homebrew launcher!
{: .notice--warning}


This also means that a cart update will NOT install a browser if you do not already have one. Isto irá **apenas** atualizar a versão do sistema *(por exemplo, 2.1.0-0 -> 6.0.0-0)*. O último número *(o -0) * **não** será alterado.
{: .notice--warning}

**Atualizar por um cartão de jogo que contenha a versão 9.9.0 ou acima irá remover o teu browser!**
{: .notice--danger}

#### O que precisas

* Um cartão de jogo que [contenha a versão](http://www.3dsdb.com/) especificada na tabela

#### Instruções

1. Liga a tua 3DS enquanto pressionas os botões L+R+A+CIMA para entrar no modo de recuperação
2. Rejeita a atualização e reinicia
3. Coloca o jogo no sistema
4. Attempt to launch the game
5. Quando pedido, atualiza a consola
